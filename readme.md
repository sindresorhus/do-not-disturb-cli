# do-not-disturb-cli

> Control the macOS `Do Not Disturb` feature from the command-line

## Install

```sh
npm install --global do-not-disturb-cli
```

## Usage

```
$ do-not-disturb --help

  Usage
    $ do-not-disturb <command>

  Commands
    on
    off
    toggle
    status

  Examples
    $ do-not-disturb on
    $ do-not-disturb status
    on

  Use `$ dnd` to quickly toggle
```
